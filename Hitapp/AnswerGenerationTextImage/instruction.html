What is this HIT about?

We are trying to build a question-answering system which is able to retrieve the answer by reasoning over multiple facts/images. This is the way we human typically use to find an answer. For example, when this question is asked:

Has there always been a rivalry between Daffy Duck and Porky Pig?

We might find the answer by

  1. Searching Wikipedia for Daffy Duck.
  2. "Daffy first appeared in Porky's Duck Hunt, released on April 17, 1937."
  3. "Porky shoots the whiskey-filled barrel but Daffy escapes" or "Daffy bites Porky on the nose"
  4. Conclude, Yes, Daffy and Porky have been rivals since their first appearance on screen.

However, for now, if you search a question in a search engine like Google or Bing, you are mostly like to get the answer from only one snippet of text. They don’t have the ability to answer the above question because, as you can see, the answer is concluded by reasoning the facts/images from multiple resources. We believe that this reasoning ability can greatly improve the question-answering ability of search engines, so we create this Hitapp and asking help from you. Appreciate your participation!

The Format:

Each hit starts with a question and group of text snippets provided by us that we think are sufficient to answer the question. These should contain all the information necessary to answer the question.. You may use these provided facts *only* to formulate your answers to the question..

Note: if you think information provided in facts is not enough to answer the question please MARK as cannot be answered

Can the information from the URL link of the fact be used in the answer?

In the following cases, the infromation from the URL link can be used in answer:

  1. If the provided information is image(s), the information in the URL link to can be used in answer.
  2. If the provided information is text snippets, the information from the URL link can be used in answer only if the fact text snippet has that information fully contained within it.
